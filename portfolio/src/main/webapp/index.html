IMU Motion Analysis

This project was conducted on behalf of my UMD Hyperloop Team

An Internal Monitoring Unit, or IMU, is a hardware set that integrates the functionality of an accelerometer, a gyroscope, and a magnetometer. Here's the problem: accelerometers and gyromescopes have a big problem when it comes to position analysis over long periods of time, because each measurement is taken in reference to the previouse position, not the original position. We wanted to use the IMU to track the forces that were being imposed on the pod as it traveled through the vacuum, and we wanted records of how it was moving. In order to do that, we need to convert the gyroscope measurements into a stream of data in world coordinates, which is then applied to the acceleration measured by the accelerometer to accurately depict direction. I programmed a C script that used quaternions on the gyroscope measurements to keep track of the changes in angles. This angular data was used with the accelerometer to get the real-time acceleration data. I utilized discrete integration to estimate the velocity.

  • Platform: None
  • Language: C, C++
  • Category: Control Systems

Honeypot

My first major project in cybersecurity

A honeypot is a threat in which an attacker, by some means, tricks the user to reveal their information. In the case of this project, the information involved would be their network and network credentials. The objective of this project was to develop a system that could protect a user from such a potential attack. The way in which it works is simple: my program creates a container environment through which all unauthorized traffic passes, and it creates a template for cloning purposes. A script is set to run in the background and listen for input. In the event that a file was added to the container, the program changes the iptables rules to kick the user that SSH'd in, kills the container, and then creates a new container based on the template that was created earlier.

  • Platform: Linux
  • Language: Bash
  • Category: Cybersecurity
